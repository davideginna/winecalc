# Guida per i Contributori

Siamo entusiasti di accogliere i tuoi contributi al progetto WineCalc! Seguendo queste linee guida, aiuti a rendere il processo di collaborazione più semplice ed efficace per tutti.

## Come Contribuire

I contributi sono benvenuti! Per contribuire:

1.  **Fai un Fork del repository:** Clicca sul pulsante "Fork" in alto a destra sulla pagina GitHub del progetto. Questo creerà una copia del repository nel tuo account GitHub.
2.  **Clona il tuo fork:** Clona il repository forked sulla tua macchina locale:
    ```bash
    git clone https://github.com/TUO_USERNAME/winecalc.git
    cd winecalc
    ```
    Sostituisci `TUO_USERNAME` con il tuo nome utente GitHub.
3.  **Crea un branch per la tua feature:** È buona pratica creare un nuovo branch per ogni funzionalità o correzione di bug.
    ```bash
    git checkout -b feature/AmazingFeature
    ```
    Dai un nome descrittivo al tuo branch (es. `feature/nuovo-calcolatore-x` o `fix/bug-interfaccia`).
4.  **Implementa le tue modifiche:** Effettua le modifiche desiderate al codice. Assicurati di seguire lo stile di codifica esistente e di testare le tue modifiche.
5.  **Commit delle modifiche:** Esegui il commit delle tue modifiche con un messaggio chiaro e conciso.
    ```bash
    git commit -m 'Aggiunto nuovo calcolatore per X'
    ```
6.  **Push al tuo branch:** Carica le tue modifiche nel tuo repository forked su GitHub.
    ```bash
    git push origin feature/AmazingFeature
    ```
7.  **Apri una Pull Request (PR):**
    *   Vai alla pagina GitHub del tuo repository forked.
    *   Vedrai un pulsante "Compare & pull request" accanto al tuo nuovo branch. Cliccalo.
    *   Fornisci una descrizione dettagliata della tua Pull Request, spiegando cosa fa la tua modifica, perché è necessaria e qualsiasi altra informazione utile.
    *   Invia la Pull Request.

## Segnalazione Bug

Se trovi un bug o hai un suggerimento:

1.  Apri una [Issue su GitHub](https://github.com/TUO_USERNAME/winecalc/issues) (sostituisci `TUO_USERNAME` con il tuo nome utente e `winecalc` con il nome del tuo repository).
2.  Descrivi il problema in dettaglio.
3.  Includi screenshot se possibile.
4.  Specifica il browser e il dispositivo utilizzato.

Grazie per il tuo contributo!
